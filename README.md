# pink_chicken

Text for AdAurum.


## Project Setup

```sh
npm install
```
### Run json-server for API mock-up
```sh
cd src/assets/data
```
```sh
npx json-server db.json
```
## Compile and Hot-Reload for Development

```sh
npm run dev
```

## Compile and Minify for Production

```sh
npm run build
```
